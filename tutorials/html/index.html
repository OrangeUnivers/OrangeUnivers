Tutorial for Csongor

<h1> bis <h6> Heading

<h1-6> auch ein "Heading" bekannt ist das Element für Titel/Untertitel. Benutze es um Kapitel oder Unterteilungen deiner Website zu zeigen.

Achtung: <h1> sollte nur einmal auf einer Seite sein weil Suchmaschinen sie zur formatierung benutzten.
